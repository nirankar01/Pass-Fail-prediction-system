# Pass-Fail-prediction-system

# 🎓 Student Performance Predictor

A simple Machine Learning web app built using Logistic Regression to predict whether a student will pass or fail based on study hours.

## 🚀 Tech Stack
- Python
- Streamlit
- Scikit-learn
- NumPy

## 📊 Features
- Predict student result (Pass/Fail)
- Simple and interactive UI
- Fast predictions using ML model

## 🧠 Model Used
Logistic Regression (Classification Algorithm)
